# Exam Topics Viewer

Simple web viewer for technical exam questions (ExamTopics-style).

This project displays exam questions, associated images, and multiple-choice answers using data generated by an external scraper.

---

## Description

The viewer is built with plain HTML, CSS, and JavaScript.
It loads questions from a JSON file and renders:

* Question text
* Optional question images (hotspot, drag & drop, etc.)
* Answer options
* Highlighted correct answers

This repository **does not perform scraping**.

## Input Data Format

Example question object:

```json
{
  "question": "Which two statements are correct?",
  "question_images": [
    "assets/images/q123_img1.png"
  ],
  "answers": {
    "A": "Option A",
    "B": "Option B",
    "C": "Option C"
  },
  "correct_answer": ["A", "C"]
}
```
